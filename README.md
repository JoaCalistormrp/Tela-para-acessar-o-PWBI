# Tela para acessar o PWBI

Sistema de login para acessar relatórios Power BI de forma segura.

## Tecnologias usadas
- HTML/CSS
- Node.js
- JavaScript

## Como executar
1. Instale as dependências com `npm install`
2. Execute o servidor com `node server.js`
3. Acesse `http://localhost:PORTA`
